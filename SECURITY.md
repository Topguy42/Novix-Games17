# Security Policy

## Supported Versions
We actively maintain the latest branch of PeteZahGames.  
Other branches may be experimental. Always deploy the stable version.

|  Branch | Supported |
|---------|-----------|
| main    | ✅        |
| others  | ❌        |

---

## Reporting a Vulnerability
If you discover a security vulnerability in this project:

1. **Do not open a public issue.**
2. Report privately using one of these methods:
   - GitHub [Security Advisories](https://docs.github.com/en/code-security/security-advisories/repository-security-advisories).
   - Email: [mailto:contact@petezahgames.com?subject=Security%20Advisory](mailto:contact@petezahgames.com?subject=Security%20Advisory).
   - Discord: https://discord.gg/cYjHFDguxS.

Please include:
- A clear description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Suggested mitigation if available.

We will acknowledge reports within 72 hours and aim to provide a fix within 30 days.

---

## Disclosure Policy
- Vulnerabilities will be disclosed publicly only after a patch is released.
- Critical issues may be patched immediately with a hotfix.
- Minor issues will be scheduled for the next release cycle.
